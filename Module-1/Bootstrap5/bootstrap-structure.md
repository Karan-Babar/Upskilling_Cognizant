# Bootstrap Structure

## CSS Folder

Contains:
- bootstrap.css
- bootstrap.min.css

Purpose:
Provides Bootstrap styling and responsive utilities.

## JS Folder

Contains:
- bootstrap.bundle.js
- bootstrap.bundle.min.js

Purpose:
Provides Bootstrap JavaScript plugins such as:
- Modal
- Accordion
- Dropdown
- Collapse

## Icons Folder

Bootstrap Icons are a separate package.

Purpose:
Provides scalable SVG icons for websites.
